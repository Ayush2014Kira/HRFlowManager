# Demo Accounts

The HRMS system comes with pre-configured demo accounts for testing and demonstration purposes.

## Available Demo Accounts

### Administrator Account
- **Username:** `admin`
- **Password:** `admin123`
- **Role:** Administrator
- **Access:** Full system access, all modules, user management, admin functions

### HR Manager Account  
- **Username:** `sarah.johnson`
- **Password:** `sarah123`
- **Role:** HR Manager
- **Access:** Employee management, leave approvals, payroll, attendance tracking

### Employee Account
- **Username:** `john.smith`
- **Password:** `john123`  
- **Role:** Employee
- **Access:** Employee portal with limited access to personal data and leave applications

## Features Available by Role

### Administrator
- Dashboard with system overview
- Complete employee management (add, edit, view, delete)
- Attendance and punch management
- Leave applications and approval workflow
- Payroll management
- User management and password reset functionality
- System reports and analytics
- Leave types and employee leave assignments
- Time tracking and field work monitoring

### HR Manager
- Dashboard with HR metrics
- Employee management
- Attendance tracking
- Leave approval workflow
- Payroll processing
- Reports and analytics
- Leave types management
- Employee leave assignments

### Employee
- Personal employee portal
- View personal information
- Apply for leaves
- View attendance records
- Access personal reports
- Time tracking (if assigned)

## Getting Started

1. Navigate to the application URL
2. Click "Login" 
3. Enter any of the demo credentials above
4. Explore the system based on your role permissions

## Note

These are demonstration accounts with sample data. In production, you should:
- Change all default passwords
- Create proper user accounts
- Configure company-specific settings
- Set up proper security policies
